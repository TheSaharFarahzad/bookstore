## Cloning the Repository
Clone this repository to your local machine using:

```bash
git clone https://github.com/your-username/bookstore.git
cd bookstore
```

## Run the Program
You can run the `main.py` file with the Python command on Windows and Linux:

```bash
python main.py
```
